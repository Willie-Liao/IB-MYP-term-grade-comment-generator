# IB-MYP Term Grade Comment Generator

An AI-powered assistant that generates personalized term report comments for IB-MYP students based on their academic performance and behavioral observations.

## Features

- Upload task clarifications for each criterion (A, B, C, D)
- Configure multiple units with custom criteria
- Upload student gradebook summaries
- AI-generated personalized report comments
- Interactive chat interface for refinement

## Setup

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in [.env](.env) to your Gemini API key:
   ```
   API_KEY=your_gemini_api_key_here
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

## How to Use

### 1. Configure Units & Criteria

After opening the interface, set up your course units:

- **Upload task clarifications** in the dropbox for each criterion (A, B, C, D)
- **Add teacher notes** in the text box underneath if needed
- If a criterion is not assessed in this unit, simply tick off **"No criterion for this unit"**
- If you have multiple units throughout the term, click **"Add another Unit"** and repeat the process

### 2. Upload Student Data

Once your units are configured:

- Upload the **student gradebook summary** generated by "Gradebook Scraper"
- The system will parse student names, scores, and assessment data

### 3. Generate Reports

- The AI will guide you through each student
- Answer questions about their behavior, attitude, submission quality, punctuality, and progress
- The system generates personalized report comments automatically
- Review and refine comments as needed through the chat interface
